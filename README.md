# TechBril
Real-time tech news web app with category filtering, search &amp; API integration (HTML, CSS, JS)
#TechBril — Live Tech & Automation News

 A real-time tech news web app delivering the latest updates in AI, automation, gadgets, cybersecurity, and science.

**Live Website:** https://bhumi6203.github.io/TechBril/
**GitHub Repository:** https://github.com/bhumi6203/TechBril

Features

Real-time tech news using Currents API
Auto-refresh every 5 minutes
Search functionality
Category filtering (AI, Tech, Gadgets, Automation, Security, Science)
Trending news section
Fully responsive design
Clean and modern UI

* HTML5

 API Configuration

```javascript
const CURRENTS_KEY = 'your_api_key_here';
```

 Get your API key from: https://currentsapi.services/


Note

* API key is used on frontend (for demo purposes)
* For production, use backend or environment variables
